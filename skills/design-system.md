# Design System

## Color Palette
- **Primary**: `#171717` (dark background)
- **On-Primary**: `#ffffff` (text on dark)
- **Ink**: `#171717` (primary text)
- **Body**: `#4d4d4d` (secondary text)
- **Mute**: `#888888` (tertiary text)
- **Hairline**: `#ebebeb` (borders light)
- **Canvas**: `#ffffff` (light background)
- **Canvas Soft**: `#fafafa` (light surface)
- **Link**: `#0070f3` (interactive elements)
- **Cyan**: `#50e3c2` (accent)
- **Violet**: `#7928ca` (accent)
- **Highlight Pink**: `#ff0080` (accent)

## Selection Effect Colors
- **Light Mode**: `#22c55e` border, `#f0fdf4` background
- **Dark Mode**: `#4ade80` border, `rgba(74, 222, 128, 0.1)` background
- **Glow**: `0 0 15px rgba(34, 197, 94, 0.4), 0 0 30px rgba(34, 197, 94, 0.2)`

## Typography
- **Font Sans**: Inter, Geist, system-ui
- **Font Mono**: JetBrains Mono, Geist Mono
- **Heading Scale**: 2xl, 3xl for questions
- **Body Scale**: base, sm, xs

## Spacing & Layout
- **Container**: `max-w-[1400px]` for nav, `max-w-[800px]` for content
- **Padding**: 6 (1.5rem) for cards, 6 (1.5rem) horizontal for pages
- **Gap**: 4 (1rem) for grids, 8 (2rem) for sections
- **Radius**: sm (6px), md (8px), lg (12px), pill (100px)

## Components
- **Cards**: `rounded-lg border p-6`
- **Buttons**: `rounded-pill px-6 py-3`
- **Indicators**: `h-8 w-8 rounded-full border`
